mdedit – Uživatelská dokumentace

mdedit je lehký desktopový editor Markdown dokumentů s přímým WYSIWYG editováním. Nevyžaduje přepínání mezi režimem editace a náhledu – formátování vidíte ihned při psaní.

✏️

WYSIWYG editace

Píšete přímo naformátovaný text, bez nutnosti zobrazovat raw Markdown syntaxi.

💾

Ukládání do .md

Dokumenty se ukládají jako čisté Markdown soubory kompatibilní s libovolným editorem.

📤

Export HTML & PDF

Jedním klikem exportujete dokument do HTML nebo PDF s profesionálním formátováním.

🖥️

Multiplatformní

Funguje nativně na Windows, macOS i Linuxu díky technologii Tauri.

Technologie mdedit je postaven na Tauri 2 (Rust backend), React 18 a Tiptap (WYSIWYG editor). Markdown soubory jsou standardní GFM (GitHub Flavored Markdown).

Prázdný stav (žádný otevřený dokument)

Pokud není otevřen žádný dokument, zobrazí se centrovaná uvítací obrazovka s logem aplikace, tlačítky Nový a Otevřít a rychlým přístupem k naposledy otevřeným souborům (max. 5 položek).

Nový dokument

Vytvoří prázdný nepojmenovaný dokument s názvem Untitled. Dokument není uložen na disk dokud nezvolíte Uložit jako….

Jak vytvořit nový dokument

  • Klávesová zkratka Ctrl+N (Windows/Linux) nebo ⌘+N (macOS)
  • Tlačítko Nový dokument (ikona FilePlus) v nástrojové liště
  • Nabídka Soubor → Nový
  • Tlačítko Nový na uvítací obrazovce
Upozornění Pokud máte otevřený dokument s neuloženými změnami, zobrazí se potvrzovací dialog. Změny budou zahozeny teprve po vašem souhlasu.

Otevření souboru

mdedit umí otevírat soubory ve formátech .md, .markdown a .txt.

Způsoby otevření

Dialogové okno systému

  • Klávesová zkratka Ctrl+O / ⌘+O
  • Tlačítko Otevřít soubor (ikona FolderOpen) v nástrojové liště
  • Nabídka Soubor → Otevřít…
  • Tlačítko Otevřít na uvítací obrazovce

Přetažení souboru (Drag & Drop)

Přetáhněte soubor .md, .markdown nebo .txt přímo na okno aplikace. Při přetahování se zobrazí modrý překryvný rámeček s textem „Drop to open file". Po puštění se soubor otevře.

Poznámka Pokud přetáhnete soubor v nepodporovaném formátu, zobrazí se chybová notifikace. Při přetažení více souborů se otevře pouze první z nich.

Otevření z příkazové řádky / „Otevřít v programu"

mdedit podporuje otevírání souborů předaných jako argument při spuštění aplikace. Funguje to při:

  • Spuštění z příkazové řádky s cestou k souboru jako argumentem
  • Systémovém „Otevřít v programu" (pravé tlačítko → Otevřít pomocí → mdedit)

Pokud je při spuštění předán soubor tímto způsobem, automatické znovuotevření naposledy editovaného souboru se přeskočí.

Uložení souboru

Uložit (Save)

Uloží aktuální dokument na stejné místo, odkud byl otevřen. Pokud jde o nový nepojmenovaný dokument, chování je stejné jako Uložit jako….

  • Ctrl+S / ⌘+S
  • Tlačítko Uložit (ikona Save) v nástrojové liště
  • Nabídka Soubor → Uložit

Uložit jako… (Save As)

Vždy otevře systémový dialog pro výběr umístění a názvu souboru. Výchozí přípona je .md.

  • Ctrl+Shift+S / ⌘+Shift+S
  • Tlačítko Uložit jako (ikona SaveAll) v nástrojové liště
  • Nabídka Soubor → Uložit jako…

Opětovné načtení ze disku (Reload)

Znovu načte aktuálně otevřený soubor z disku a zahodí všechny neuložené změny. Tlačítko je dostupné pouze tehdy, pokud byl soubor dříve uložen.

  • Ctrl+Alt+R / ⌘+Alt+R
  • Tlačítko Obnovit ze disku (ikona RefreshCw) v nástrojové liště
  • Nabídka Soubor → Reload from disk

Indikátor neuložených změn

Pokud máte neuložené změny, zobrazuje se:

  • Hvězdička (*) v záhlaví okna vedle názvu souboru
  • Text Modified (oranžově) ve stavovém řádku

Po uložení se zobrazí zelená notifikace „Saved" a indikátor zmizí.

Export dokumentu

mdedit umožňuje exportovat dokument do dvou formátů, přístupných z nástrojové lišty nebo nabídky Soubor.

Export do HTML

Vytvoří samostatný HTML soubor se vloženým CSS stylingem. Výsledný soubor lze otevřít v jakémkoliv webovém prohlížeči bez dalších závislostí.

  • Tlačítko Export HTML (ikona FileCode) v nástrojové liště
  • Nabídka Soubor → Export as HTML…

Export do PDF

Vytvoří tiskově připravený PDF soubor s více stránkami, správnými okraji a zalamováním stránek. PDF je generováno z interního HTML rendereru, takže vizuálně odpovídá HTML exportu.

  • Tlačítko Export PDF (ikona FileDown) v nástrojové liště
  • Nabídka Soubor → Export as PDF…
Tip Navrhovaný název exportovaného souboru se automaticky odvozuje od názvu aktuálního dokumentu.

Poslední soubory

mdedit si pamatuje posledních 10 otevřených souborů. Přístup k nim je možný třemi způsoby:

  • Rozbalovací nabídka Poslední soubory v nástrojové liště (ikona šipky dolů)
  • Nabídka Soubor → Recent Files (podnabídka)
  • Uvítací obrazovka (zobrazuje posledních 5 souborů)

Soubory, které již na disku neexistují, jsou ze seznamu automaticky odstraněny.

Automatické znovuotevření

Při spuštění aplikace se automaticky otevře naposledy editovaný soubor. Toto chování je implicitně zapnuto a lze jej změnit v nastavení aplikace.

Formátování textu

Formátování lze aplikovat pomocí tlačítek v nástrojové liště nebo klávesovými zkratkami. Formátování funguje jak na vybraný text, tak jako přepínač pro nově psaný text.

Tučný text (Bold)

Zobrazí text tučně. V Markdown se ukládá jako **text**.

  • Klávesová zkratka: Ctrl+B
  • Tlačítko B v nástrojové liště

Kurzíva (Italic)

Zobrazí text kurzívou. V Markdown se ukládá jako *text*.

  • Klávesová zkratka: Ctrl+I
  • Tlačítko I v nástrojové liště

Inline kód

Zvýrazní část textu jako kód ve fontu monospace. V Markdown se ukládá jako `kód`.

Lze použít psaním backtick znaku (`) přímo v editoru.

Nadpisy

Nadpisy se spravují pomocí tlačítka H v nástrojové liště, které cyklicky přepíná mezi úrovněmi:

Aktuální stavPo kliknutí na HMarkdown
Normální odstavecNadpis H1# Nadpis
H1H2## Nadpis
H2H3### Nadpis
H3Normální odstavec(bez prefixu)

Aplikace podporuje nadpisy úrovní H1 až H6, přičemž přímý cyklus v toolbaru pokrývá H1–H3.

Seznamy

Odrážkový seznam (Bullet List)

Neodrážkový seznam s odrážkami. V Markdown se ukládá s prefixem -.

  • Klávesová zkratka: Ctrl+Alt+U
  • Tlačítko seznam (ikona List) v nástrojové liště

Číslovaný seznam (Ordered List)

Seznam s automatickým číslováním položek.

  • Klávesová zkratka: Ctrl+Alt+Shift+U
  • Tlačítko číslovaný seznam (ikona ListOrdered) v nástrojové liště

Seznam úkolů (Task List)

Interaktivní seznam s zaškrtávacími políčky. Políčka lze zaškrtnout přímo kliknutím. V Markdown se ukládá jako - [ ] úkol nebo - [x] splněno.

  • Tlačítko Task List (ikona CheckSquare) v nástrojové liště

Chování task listu

  • Při vybraném textu: Každý řádek výběru se transformuje na samostatnou položku task listu.
  • Při prázdném výběru: Vloží se checkbox na aktuální řádek a editor přejde do režimu task listu.
  • Enter: Vytvoří další položku s checkboxem.
  • Prázdný Enter: Ukončí seznam (stejné chování jako u bullet/ordered listu).
  • Tab / Shift+Tab: Odsazení a zrušení odsazení položky.

Příklad transformace textu na task list

Před (vybraný text):
řádek 1
řádek 2

Po (task list):
- [ ] řádek 1
- [ ] řádek 2

Tabulky

Vloží výchozí tabulku 3×2 (3 sloupce, 2 řádky) s řádkem záhlaví. Buňky lze volně editovat kliknutím, přecházet mezi nimi klávesou Tab.

  • Tlačítko Tabulka (ikona Table) v nástrojové liště

Tabulky jsou ukládány v GitHub Flavored Markdown formátu:

| Sloupec 1 | Sloupec 2 | Sloupec 3 |
|-----------|-----------|-----------|
| Buňka     | Buňka     | Buňka     |

Kód a bloky kódu

Inline kód

Označení části textu jako kódu uvnitř odstavce. Použijte backtick (`) nebo příslušnou Markdown syntaxi.

Blok kódu (Code Block)

Vloží víceřádkový blok kódu s monospacovým fontem a zvýrazněním syntaxe.

  • Tlačítko Kód (ikona Code) v nástrojové liště

Výběr jazyka

Když je kurzor uvnitř bloku kódu, aktivuje se v nástrojové liště rozbalovací nabídka pro výběr programovacího jazyka. Výběr jazyka ovlivňuje zvýraznění syntaxe.

Odkazy a obrázky

Vložení odkazu (Link)

Otevře dialog pro zadání URL adresy odkazu.

  • Tlačítko Odkaz (ikona Link) v nástrojové liště
  • Pokud URL nezačíná schématem (http://, https://, #, /), automaticky se předřadí https://
  • Podporovány jsou relativní cesty a kotvy (#sekce)
  • Výsledné odkazy se otevírají v novém záložce (target="_blank")

Odebrání odkazu

Odstraní odkaz z vybraného textu, text zůstane zachován.

  • Tlačítko Odebrat odkaz (ikona Unlink) v nástrojové liště

Vložení obrázku (Image)

Otevře dialog, kde zadáte URL obrázku a volitelně alternativní text (popis pro přístupnost).

  • Tlačítko Obrázek (ikona Image) v nástrojové liště
Poznámka mdedit v současné verzi pracuje s URL obrázků (vzdálené nebo lokální cesty). Vkládání obrázků přetažením přímo do editoru není podporováno.

Ostatní blokové prvky

Citace (Blockquote)

Blok citace se zvýrazněným levým okrajem. V Markdown se zapisuje prefixem >.

Horizontální oddělovač

Vodorovná čára oddělující části dokumentu. V Markdown se zapisuje jako ---.

Vizuální stavy tlačítek

Tlačítka v nástrojové liště mají tři vizuální stavy:

  • Normální: Výchozí vzhled
  • Aktivní: Modrý okraj + pozadí (formátování je aplikováno nebo kurzor je uvnitř prvku)
  • Zakázané: 40% průhlednost (akce není v daném kontextu dostupná)

Všechna tlačítka mají tooltip s popisem akce a klávesovou zkratkou.

Panel osnovy

Panel osnovy (Outline) se zobrazuje na levé straně editoru a poskytuje přehled struktury dokumentu na základě nadpisů.

Funkce panelu osnovy

  • Hierarchické zobrazení: Nadpisy jsou odsazeny podle úrovně (H1 bez odsazení, každá další úroveň přidává odsazení).
  • Navigace: Kliknutím na položku skočíte na příslušný nadpis v dokumentu.
  • Aktivní nadpis: Aktuálně viditelný nadpis je zvýrazněn.
  • Prázdný stav: Pokud dokument neobsahuje žádné nadpisy, zobrazí se zpráva „No headings yet".
  • Okamžitá aktualizace: Osnova se aktualizuje ihned po otevření dokumentu i při každé editaci.

Zobrazení / skrytí panelu

  • Tlačítko PanelLeft v nástrojové liště
  • Nabídka Zobrazení → Toggle Outline

Stav panelu (zobrazeno/skryto) se ukládá a zachovává mezi restarty aplikace.

Stavový řádek

Stavový řádek ve spodní části okna zobrazuje informace o aktuálním dokumentu.

Levá část – název souboru

  • Název aktuálně otevřeného souboru
  • Untitled pro nový nepojmenovaný dokument
  • No document pokud není otevřen žádný soubor

Pravá část – stav

  • Saved (šedě): Dokument je uložen, žádné neuložené změny
  • Modified (oranžově): Dokument má neuložené změny
  • External change detected (oranžově): Soubor na disku byl změněn jinou aplikací – zobrazí se výzva k obnovení

Notifikace (Toast)

Aplikace zobrazuje informační notifikace v pravém dolním rohu obrazovky. Notifikace se automaticky skryjí po několika sekundách, nebo je lze zavřít ručně kliknutím na ×.

Dialogy

Potvrzovací dialogy

Zobrazují se před nevratnými nebo rizikovými operacemi:

  • Zahození neuložených změn (při otevření nového souboru nebo vytvoření nového dokumentu)
  • Zavření aplikace s neuloženými změnami

Klávesové zkratky v dialogu:

  • Escape – Zrušit (cancel)
  • Enter – Potvrdit (pokud je povoleno)

Vstupní dialogy

Používají se pro zadání textu uživatelem:

  • Vložit odkaz: Pole pro zadání URL (ověřuje se neprázdnost)
  • Vložit obrázek: Nejprve URL obrázku, poté volitelný alternativní text

Vstupní pole se při otevření automaticky zaměří a obsah se vybere. Validační chyby se zobrazují přímo pod polem.

Normalizace Markdownu při ukládání

Při uložení souboru mdedit automaticky normalizuje Markdown:

  • Konce řádků jsou sjednoceny na LF (\n)
  • Bílé znaky na konci řádků jsou odstraněny
  • Více prázdných řádků za sebou se sjednotí na jeden
  • Zaškrtnuté úkoly se normalizují ([X] → [x])
  • Na konci souboru je přesně jeden prázdný řádek
  • Obsah kódových bloků zůstává beze změny

Detekce externích změn

mdedit sleduje, zda byl aktuálně otevřený soubor změněn jinou aplikací (např. textovým editorem, verzovacím systémem apod.).

Pokud je detekována zewnętrzní změna:

  • Ve stavovém řádku se zobrazí oranžový text „External change detected"
  • Tlačítko Obnovit ze disku v nástrojové liště se vizuálně zvýrazní

Pro načtení aktuální verze souboru z disku použijte klávesovou zkratku Ctrl+Alt+R nebo klikněte na tlačítko Obnovit.

Upozornění Obnovení ze disku zahodí všechny vaše neuložené změny v editoru.

  • Reload from disk Ctrl+Alt+R (zakázáno, pokud soubor nebyl uložen)
  • Export as HTML…
  • Export as PDF…

  • Quit (ukončit aplikaci)

Úpravy (Edit)

  • Undo / Redo
  • Cut / Copy / Paste
  • Select All

Zobrazení (View)

  • Toggle Outline (zaškrtávací položka odrážející stav panelu osnovy)
