mdedit

Odlehčený multiplatformní Markdown editor.
Píšte, co vidíte – bez přepínání mezi editací a náhledem.

Stáhnout Zdrojový kód

v0.1.0 · MIT licence · Tauri + React + TypeScript

Co mdedit umí

✍️

WYSIWYG editing

Výsledek vidíte přímo při psaní. Žádné přepínání mezi zdrojovým kódem a náhledem.

📁

Práce se soubory

Otevírání, vytváření a ukládání .md, .markdown a .txt souborů. Drag & drop, CLI argumenty i „Otevřít v".

🗂️

Osnova dokumentu

Postranní panel s přehledem nadpisů umožňuje rychlou navigaci i v delších dokumentech.

🔧

Panel nástrojů

Tučné písmo, kurzíva, nadpisy, seznamy, úkoly, tabulky, odkazy, obrázky, bloky kódu – vše na dosah kliknutí.

💾

Sledování změn

Indikátor neuložených změn v titulku okna a ochrana před nechtěným zavřením bez uložení.

📤

Export

Dokument lze exportovat do HTML nebo PDF přímo z aplikace.

🕐

Poslední soubory

Přehled naposledy otevřených souborů a automatické otevření posledního dokumentu při startu.

🖥️

Multiplatformní

Nativní desktopová aplikace postavená na Tauri – běží na Linuxu, macOS i Windows.

AkceZkratka

Technologie

  • Desktop shell Tauri 2
  • Frontend React 18 + TypeScript
  • Editor engine Tiptap 2
  • Markdown pipeline unified / remark / rehype
  • State management Zustand 4
  • Build tool Vite 5
  • Backend Rust

Pro vývojáře

Zkompilovat svépomocí

Sestavit aplikaci ze zdrojového kódu zvládnete v pár krocích. Předpoklady: Node.js 18+ a Rust stable.

Zdrojový kód na GitHubu 
git clone https://github.com/MartinDejmal/mdedit.git
cd mdedit
npm install
npm run tauri build

Hotový instalátor po buildu najdete v src-tauri/target/release/bundle/.
